User interface related options.

GUI Language

Select GUI language. If you leave "From Locales" selected, it will search for the locales of your operating system. You can override this option for one session by command line option.

This option will take effect after application restart.

GUI Style

You can specify the application look here. Qt4 library allows you to use GUI "skins". I suggest to use the theme to fit your desktop.

This option will take effect after application restart.
