The Result Table

A place to present the results of your statements.

  1. Insert new row. If you are inserting multiple rows, you must provide at least one attribute for each row. The data after the empty row will be skipped.
  2. Delete current row.
  3. Delete all from this table.
  4. Commit all data result table related changes.
  5. Rollback all data result table related changes.
  6. Export data to the file in the chosen fromat.
  7. Show data.
  8. Message area - database errors, SQL syntax errors etc.

Data Export

Format

Choose requested format. Supported amount of formats can be platform different. Depending on the Open Source libraries availability.

Export to:

File – select a file to save by the "Search" button.

Clipboard – export data into your system clipboard to use in the others application directly. Note: if you are exporting the huge amount of data your system can be slowed down rapidly. Use it at your own risk.
